# 刷题复盘台

一个用于管理机试刷题代码、复试口述思路和复习状态的本地网页工具。项目是纯静态页面，不需要安装依赖，直接打开 `index.html` 即可使用。

## 页面结构

- 个人主页：展示题目总数、今日新增、待复习、薄弱题，并提供近一年添加热力图、状态分布、近期动态和复习建议。
- 提交页面：录入题号、题名、链接、难度、语言、标签、题目描述、核心思路、易错点和代码；可通过 LeetCode 链接尝试自动抓取题目信息。
- 题库记录：支持关键词、分类、状态筛选，并可在“部分代码”和“所有代码”之间切换。
- 复习页面：随机抽查题目，先隐藏代码进行口述复盘，再按掌握程度标记为“掌握 / 模糊 / 不会”。

## 数据保存

数据保存在浏览器 `localStorage` 中。更换浏览器、清理浏览器数据或使用隐私模式可能导致记录丢失，建议定期使用页面左侧的“导出 JSON”备份。

## 快速开始

直接用浏览器打开：

```text
index.html
```

Windows 下也可以双击 `index.html` 使用。

## 项目文件

```text
.
├── index.html   # 页面结构
├── styles.css   # 页面样式
├── app.js       # 数据、筛选、导航、复习和导入导出逻辑
└── README.md    # 项目说明
```

## 推荐使用流程

1. 每写完一道题，在“提交页面”保存代码、思路和易错点。
2. 在“题库记录”按标签或状态筛选，快速回看所有代码或部分代码。
3. 每天先看“个人主页”的待复习和薄弱题。
4. 进入“复习页面”抽查，先口述解法，再显示代码对照。
5. 按结果标记掌握程度，让后续抽查更偏向薄弱题。
